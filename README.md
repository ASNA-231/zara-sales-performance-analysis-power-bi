# zara-sales-performance-analysis-power-bi

Insurance Risk & Claims Analysis Dashboard

This project analyzes Insurance policies and claim data to identify major risk patterns and high claim segments. The dashboard is built using Power BI.

Key Insights

Total Policies: 38K

Total Claim Amount: $187.8M

Avg Claim Amount: $5.0K

Claim frequency is almost equal across genders

Highest claim amounts seen in:

Age Group 56-65

Private Car Use

Bachelors Education Category

Tool Used

Power BI Desktop

Excel dataset (source data cleaning + preprocessing)

Files Included

Dataset: insurance_policies_data.xlsx

Insurance Dashboard PDF output

What this project shows

Claim Patterns by Age, Kids, Car Use, Car Make, Car Year, Education & Zone

Risk segments that affect insurance claim behavior
